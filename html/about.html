A personal calendar is provided for every user. To this calender, you can add filters for things like work, class schedules, or any filters you would want.

To create a new filter, simply click "add filter" button on the "profile" page. You can add events to various filters within the calender view by clicking the "+" button with the desired filters selected.

Filters are a secure way, to let friends family and coworkers access your daily tasks, filters come with privacy that could be set according to the users needs.

To find friends who use instaCal, just search for them by username/email in the friends tab

Only users who have been given access the filters will be able to see the events in a specific filter
